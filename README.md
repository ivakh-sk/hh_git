# HH Auto Apply

Автоматизированный скрипт для поиска и отклика на вакансии на hh.ru  
с использованием HH API и Playwright.

Проект предназначен **исключительно в учебных и демонстрационных целях**.

---

## Как работает

1. Поиск вакансий через HH API
2. Фильтрация:
   - по зарплате
   - по опыту
   - по названию вакансии
   - по типу работодателя
   - по формату работы (remote / hybrid)
3. Автоматический отклик через браузер (Playwright)
4. Логирование результатов в CSV-файл

---

## Требования

- Python **3.10+**
- Аккаунт на hh.ru
- Установленный браузер Chromium (ставится через Playwright)
